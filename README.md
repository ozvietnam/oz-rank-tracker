# OZ Rank Tracker

**Theo doi tu khoa SEO cho thutucxuatnhapkhau.com**

Dashboard quan ly rank keywords — single-file React app, localStorage persistence, mobile-friendly.

## Tinh nang

- Them / xoa / update rank keywords
- Metrics: Top 10, Top 20, Total, Tracked
- Trend indicator
- Color-coded badges (xanh top10, vang top20, do below)
- Responsive (dien thoai, tablet, desktop)
- Luu data trinh duyet (khong can dang nhap)

## Deploy

App nay la **static HTML** — co the deploy tren Vercel, Netlify, GitHub Pages.

### Vercel (khuyen dung)
Import repo nay vao https://vercel.com/new/import → Deploy (framework: Other).
